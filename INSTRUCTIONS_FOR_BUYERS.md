## In 3 steps

1. **Install and run**  
   In the project folder run: `npm install` then `npm run dev`. Open http://localhost:3000.

2. **Edit one file**  
   Open **`src/data/company_data.js`** and update your company name, industry, tagline, nav, stats, services, case studies, testimonials, contact form text, and footer. Save and refresh the browser.

3. **Deploy**  
   Push to GitHub and connect the repo to Vercel (or use Netlify). Set **siteUrl** in `company_data.js` to your live URL. Optionally wire the contact form to email (see DEPLOYMENT.md).

## What you get

- Next.js 14 + TypeScript + Tailwind CSS
- Single config file for almost all content (`company_data.js`)
- Sticky navbar, hero, stats bar, services, case studies, trust bar, testimonials, about, insights, contact form, footer
- Contact form with inquiry type (General, RFP, Partnership, Enterprise)
- Optional cookie consent bar
- SEO: JSON-LD, Open Graph, meta tags
- Responsive, mobile-friendly layout
- Placeholder pages for Privacy and Terms (you add the real text)

For template setup and customization, use the docs above. For licensing or purchase support, contact the seller from the store where you bought the template.
